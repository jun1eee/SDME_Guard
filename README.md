# 스드메의 문단속

> AI 웨딩 어시스턴트 기반 스드메 직거래 플랫폼

## 프로젝트 배경

결혼 준비에서 스드메(스튜디오·드레스·메이크업)는 평균 520만 원에 달하지만, 불투명한 가격 구조와 웨딩플래너의 중간 수수료(30~40%)로 인해 실제 비용은 더 높아진다. 예비부부는 정확한 비용을 사전에 알지 못한 채 계약하는 경우가 대부분이며, 소비자 피해 상담 건수는 2021년 790건에서 2023년 1,293건으로 급증하고 있다.

2025년 정부는 가격 공개 의무화, 표준계약서 제정 등 웨딩 시장 투명화 정책을 본격 시행하고 있다. 이 흐름에 맞춰 **웨딩플래너를 AI로 대체**하고, 예비부부와 업체가 **직접 거래**할 수 있는 플랫폼을 만든다.

## 해결하려는 문제

| 문제 | 현황 |
|------|------|
| 불투명한 가격 | 기본 패키지 외 추가금이 계약 후 발생, 최종 금액 예측 불가 |
| 높은 중간 수수료 | 웨딩플래너가 업체 비용의 30~40%를 수수료로 수취 |
| 정보 비대칭 | 예비부부에게 가격 비교 수단이 없고 업체 정보에 종속 |

## 핵심 기능

### AI 웨딩 어시스턴트
- 예산·시기·지역 기반 맞춤 업체 추천
- D-day 기반 일정 관리 및 리마인더
- 계약서 불공정 조항 자동 감지
- 성수기/비수기 시세 안내 및 추가금 사전 경고

### 스타일 매칭 AI
- 레퍼런스 사진 업로드 시 CLIP 기반 촬영 스타일 분석
- 업체 포트폴리오와 유사도 계산 후 매칭 추천
- 스타일 기반 추가금 예측

### 투명한 가격 비교 & 직거래
- 업체별 기본가·추가옵션 가격 전면 공개
- 실제 이용자의 최종 결제금액 비교 표시
- 사진 인증 리뷰 및 추가금 발생 내역 공유

### 인증 리뷰 시스템
- 거래내역 기반 인증 리뷰 (내부 결제 자동 연동 / 외부 영수증 OCR 검증)
- 항목별 만족도 평가 (결과물 / 추가금 / 응대 / 촬영 분위기)
- AI 리뷰 요약

## 기술 스택

| 영역 | 기술 |
|------|------|
| AI 상담 | LLM + RAG |
| 스타일 분석 | CLIP |
| 추천 | Learning-to-Rank, Bandits |
| 일정 관리 | MCP, Google OR-Tools |
| 3D 드레스 미리보기 | SAM 3D, TripoSR |

## 대상 사용자

- **예비부부** (20~35세) — 결혼 준비 비용 절감 및 투명한 정보 확보
- **스드메 업체** — 플래너 수수료 없이 직접 고객 확보

## 기대 효과

- 스드메 비용 **30~40% 절감** (플래너 수수료 제거)
- 가격 투명화로 추가금 사전 예측 가능
- 정부 투명화 정책과 시너지
